WHAT DOES A WEB DESIGNER DO?


Web designers plan, create and code internet sites and web pages, many of which combine text with sounds, pictures, graphics and video clips. A web designer is responsible for creating the design and layout of a website or web pages. developing concepts, graphics and layouts, including making decisions about fonts, images, readability and readers' needs. determining size and arrangement of copy and illustrative material, as well as font style and size. preparing rough drafts of material based on an agreed brief.

It and can mean working on a brand new website or updating an already existing site. Their duties include coding webpages or entire websites, meeting with clients to review website templates or refine their designs and running tests to preview layouts and website features. it involves two major skill sets: web design and web development. Web design determines the look and feel of a website, while web development determines how it functions.
